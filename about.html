THE PROJECT

Data+AI is a free tutorial website aimed at teaching the basics of data analysis while utilising simple AI to optimise your workflow. It is aimed at beginners and will break down the entire process into 5 stages:

  1. Plan
  2. Prepare
  3. Process
  4. Analyse
  5. Share

This process is based on the material from the Google Data Analytics Professional Certificate by Coursera. This is a popular course due to its industry-standard approach.

An example project will be used to demonstrate all the necessary skills. This will be a musical trend analysis using Kanye West lyrics and sales figures. However, feel free to pick your own artist or project. You can find other datasets by searching or viewing trending and popular Kaggle datasets. For example, here's a dataset of Taylor Swift Song Lyrics. You can browse other analyst's projects at Kaggle Notebooks.

By the end of the project, you will be able to predict the success of an album by determining the most frequently used words in each track. You will learn to create word clouds as an effective visualisation for text data. This website is a jumping off point to learn these skills and then take them further on your own projects. It is a resource you can return to in any order to refresh your memory.

A Data Analyst is a rewarding and sought-after career and the more prior education you have will allow you to build an enticing portfolio that demonstrates your personal skills to prospective employers. Even if you are not seeking a career in this field directly, many jobs require conducting rudimentary data analysis to make sound business decisions.

It is assumed you are familiar with the concepts of data analysis and artificial intelligence, along with essential computer skills such as downloading programs and using chat bots. If you don't feel comfortable with that, I recommend following W3School's Python Tutorial and playing around with OpenAI's ChatGPT to get started.

TECHNOLOGY

Many of you may be familiar with data analysis through the use of spreadsheet programs such as Microsoft Excel. However, in the current information age, the amount of data created and consumed is increasing exponentially. Many companies now utilise big data in their analysis. In order to process large amount of information more efficiently, coding is used to simplify and automate the process. AI is used to streamline this process even further. For these reasons, this website will focus on the following software:

  • Python - programming language
  • Jupyter Notebook - interactive projects
  • Visual Studio Code - coding environment
  • ChatGPT - AI chatbot
  • Matplotlib - Python visualisation library
  • Pandas - Python data analysis library
